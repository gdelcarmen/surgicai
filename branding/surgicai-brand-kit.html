01 — Logo System
Primary Logo — Dark Hero / Landing / Pitch

When to use

Homepage hero, pitch decks, conference slides, splash screens, app store listings. Always on dark backgrounds. The suture element should only appear in this primary context.
Wordmark — Dark Default

When to use

Nav bars, headers, documentation, in-app branding, email signatures, footers. No suture — the gradient AI carries the identity alone.
Wordmark — Light White / Light BG

Light background rule

On white or light backgrounds: wordmark only, no suture. "surgic" shifts to near-black, gradient deepens slightly for contrast.

05 — Animated Suture (Loading)
Infinite Threading — Full Width Loading / Splash

Infinite Threading

Suture stitches scroll infinitely left-to-right on a wound line beneath the wordmark. The gradient repeats seamlessly. Use for splash screens, loading states, page transitions.
Thread Draw-On Intro / Boot
(refreshes every 4s)

Draw-On Sequence

Wound line draws itself, then stitches pop in one by one left-to-right. Perfect for initial app boot or first-time experience. Plays once — doesn't loop.
Breathing Pulse Subtle / Thinking

Breathing Pulse

Stitches fade in and out in a staggered wave — like a pulse moving through the suture. Subtle and ambient. Use for "AI is thinking" states, background processing, or ambient branding.
Suture Loading Bar — Compact Inline Loading
RETRIEVING SOURCES...
Synthesizing...

Compact Loading

The suture pattern at two compact sizes — usable as inline loading indicators. "RETRIEVING SOURCES..." for MARCUS queries, "Synthesizing..." for AI processing. Replace any generic spinner.
06 — Usage Rules
✓ Do
Use primary logo (with suture) on dark backgrounds only
Use bare wordmark on light backgrounds
Maintain coral gradient on "AI" in all contexts
Use SAI icons at 16px+ for favicons
Use suture loading animation for AI processing states
Keep minimum clear space of 1x height around logo
Use DM Sans for all UI text, JetBrains Mono for code/labels
✗ Don't
Show suture on white or light backgrounds
Add a dot between "surgic" and "AI"
Use the logo in a single flat color
Rotate, skew, or distort the wordmark
Place on busy backgrounds or images
Add effects (drop shadow, outer glow, bevel)
Stack "surgic" over "AI" — always horizontal
